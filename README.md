# Portfolyo-Sitesi
Okul projesi için yapmakta olduğum web projesi



## 🎯 Özellikler Detayı

### Proje Galerisi
- IDA Interface projesi
- Air Defence projesi
- Cloud Photo projesi
- IHA Simulation projesi

### Blog Sistemi
- MDX ile içerik yönetimi
- Gemini AI ile otomatik içerik üretimi
- Kategori bazlı filtreleme
- Arama özelliği

### İletişim Formu
- Form doğrulama
- Email gönderimi
- Sosyal medya bağlantıları
